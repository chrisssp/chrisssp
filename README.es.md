### 👨‍💻 Sobre mí

Soy un Ingeniero de Software especializado en **Desarrollo Web y Móvil** con sistemas **Backend** robustos. Me desenvuelvo en entornos Linux y priorizo la escritura de código mantenible, escalable y de alto rendimiento.

* 🔭 Actualmente trabajo en **soluciones impulsadas por IA** para finanzas conductuales y salud.
* 🐧 Desarrollo en **Linux**.
* ⚡ Metodología: Practicante de **Scrum**, **Kanban** y **Design Thinking**.

### 💼 Impacto Profesional

#### 🏗️ [7D-Compass](https://7d-compass.com/) (Seven D Construction)
* **Impacto:** Automatización de conciliación financiera reduciendo el tiempo de procesamiento manual en un **95%** (de 12h a menos de 30m semanales).
* **Escala:** Gestión y auditoría de **\$2.3M USD** en transacciones con lógica de validación personalizada.
* **Tecnologías:** Angular, Node.js (Express), PostgreSQL (PostGIS).

### 🚀 Proyectos Destacados

#### 🐢 [Azkali](https://azkali-landing.vercel.app/) (Talent Land 2026)
> *Copiloto de IA conductual diseñado para mitigar gastos impulsivos en la Gen Z.*
* **Hackathon:** Construido en un **sprint de 48 horas** en Talent Land CDMX.
* **Innovación:** Análisis de impulsividad en tiempo real y traducción a costo de oportunidad operativo.
* **Tecnologías:** React Native, Gemini 2.5 Flash-Lite, FastAPI, Supabase.

#### 🏥 [IAPEX](https://github.com/aescobar80/IAPEX_APP-WEB) (IA para Localización de Pacientes)
> *Inteligencia Artificial para la localización de pacientes extraviados en instituciones de salud.*
* **Impacto:** Permite a los hospitales identificar pacientes vulnerables rápidamente mediante fusión de información híbrida.
* **Tecnologías:** Angular, Ionic, Spring Boot, Django (AI), PostgreSQL.

#### 🩺 [dabetai](https://github.com/dabetai-org/dabetai-mobileapp)
> *Plataforma preventiva para diabetes que conecta wearables con supervisión médica.*
* **Tecnologías:** React Native, NestJS, Expo, Python, Wearables.

#### 💳 [Punto Fiel](https://github.com/chrisssp/puntofiel-mobileapp)
> *Plataforma de fidelización para negocios locales con recompensas mediante códigos QR.*
* **Tecnologías:** React Native, Expo, Supabase (RLS), TanStack Query.
